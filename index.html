The Quick & Dirty of HTML

What is HTML?

HTML stands for Hypertext Markup Language. Hypertext refers to text that, when selected, can take you to another webpage. Markup is a set of instructions that specifies how a web page should be displayed in a web browser. Language in this case refers to the set of codes that comprise the markup of a page

Markup codes called tags specify the structure of the webpage. These act as containers for text and images.

Tag Attributes

Tags can be modified for specificity that determines size, style or function. By using tags such as class, id, type and href, you lay out a structure that can be easily stylized.

Basic Structure

Every web page will begin with a Doctype declaration that tells the browser that this is an HTML page.

This appears as <!DOCTYPE html>

Following this is the line <html lang="eng">, which tells the web browser two things: the type of file (HTML), as well as a language attribute (English).

After these declarations, the head and the body are the first level of organization for HTML. The head will feature other declarations, such as the font style, symbols that can be used, associated style sheets and the page's title. A title allows the page to be found in a web search

The body of the page is what will actually be seen by the user - text, images, and other graphics. The body will be further subdivided with tags that help to structure the layout of the page.

Structure v. Style

It is important to separate the structure of a web page from its styling. HTML provides the structure, and CSS handles the styling. Structure is broken down to the level of sentences, words, and even characters, in addition to blocks that may be immediately apparent.

Some HTML tags already have predefined styling, that one should be aware of. However, even these elements can be further stylized using CSS.

Text FX

HTML has a few tags that add structure to text, but which should be based on the meaning you wish the text to convey

For example, if you wish to put emphasis on a particular word so that the user reads it as such, you will use the tag <em>. Text will be rendered in italics

To mark important text that you don't want readers to miss, you might want it to appear in bold. This is done by using the tag <strong>.

When dividing your page into sections, sometimes it is desirable to use headings. HTML uses various tags in order to display text that may be bigger or appear in bold typeface. There are six HTML heading tags, starting with <h1> and ending with <h6>.

Creating Links

This is where hypertext comes in. Creating links can direct a user to another part of your web page or to external pages.

The tag associated with links is <a>. An attribute within this tag is href="URL". Descriptive link text is placed between the anchor tags.

So for example, this line of code:

<a href="https://html.com/">html.com</a>

will take you to html.com.

To link to another part of your own web page, the href will be a # followed by the id of that section of the page.

Make a List

Another element that is structural but also stylistic is a list. With HTML you can create bulleted and numbered lists. These are know as unordered lists (<ul>) and ordered lists (<ol>).

An unordered list consists of the <ul> tag and list items with the tag <li> are nested within. So a bulleted, unordered list such as:

Foods I like:

  • Dark chocolate
  • Cheddar cheese
  • Strawberries

will be coded in this manner:

<p>Foods I like:</p>
<ul>
<li>Dark chocolate</li>
<li>Cheese</li>
<li>Strawberries</li>
</ul>

A numbered, ordered list such as:

Foods I don't like:

  1. White chocolate
  2. Bleu cheese
  3. Papaya

will be coded like this:

<p>Foods I don't like:</p>
<ol>
<li>White chocolate</li>
<li>Bleu cheese</li>
<li>Papaya</li>
</ol>

Notice that the numbers are automatically assigned simply by using the ol tag.

Using Special Characters

In the head of the document, the line <meta charset="utf-8"> appears and signifies that you are using the UTF character set. This means that you are able to insert any character in any language by using a combination of keyboard strokes and specific codes.

Adding Images

To give the page more visual interest, you can insert images in one of threee types: GIF, JPEG and PNG. The <img> tag uses the attribute of <src> (the image source) and <alt> (descriptive text that is used for accessibility).

An HTML Skeleton

A high level of structure in HTML documents is imperative in order to be user-friendly. Just as the words you are reading now are separated and made intelligible with spaces between words, punctuation between sentences, and blocks of text such as paragraphs, tags are used to define the layout of a web page.

Tags such as <h2>, <nav>, <main>, <article>, <section>, <aside>, and <footer> can divide the page into containers that help to divide content in a logical way.
